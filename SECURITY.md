# Security Policy

## Reporting a vulnerability

If you discover a security issue in this repository, please report it privately via GitHub Security Advisories for [xgic/gitlab](https://github.com/xgic/gitlab) when available, or contact the maintainers through the organization listed at [github.com/xgic](https://github.com/xgic).

Do **not** open a public issue with exploit details or secrets.

## Public-safe contributions

This is a public exemplary repository. Do not include private infrastructure details, private tracker IDs, internal paths, or credentials in issues, PRs, or commits.

Multi-repo standards: https://github.com/xgic/ai
